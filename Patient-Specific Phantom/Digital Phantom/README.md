## Patient-Specific Phantom

Two types of patient-specific phantoms were evaluated in this project: a **digital phantom** and a **HU-based phantom**. The HU-based approach directly incorporates CT images into MC-GATE simulations using calibration files that convert Hounsfield Units (HU) into tissue properties.

The following code was developed:

- **[Generate_SegmentationsLabel_3DSlicer.txt](./Generate_SegmentationsLabel_3DSlicer.txt)** – Custom *3D Slicer* script that generates a file listing all segmented structures and their corresponding label values. Each voxel within the defined VOIs is assigned a unique label, enabling the subsequent definition of material compositions and densities in GATE.

- **[Generate_GATE_range_translator.py](./Generate_GATE_range_translator.py)** – Python script that uses the file from Generate_SegmentationsLabel_3DSlicer.txt to automatically generate a GATE compatible range translator file. It reads the exported labelmap information and links each segmentation label to its corresponding material definition, formatting the output according to GATE requirements.
